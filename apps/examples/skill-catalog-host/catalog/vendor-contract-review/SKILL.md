# Vendor Contract Review Skill

When installed, review vendor agreements for:

- renewal and termination dates
- payment terms
- data handling obligations
- assignment and change-of-control limits
- unusual indemnity, liability, or exclusivity terms

This sample is marked as paid metadata, but the example host intentionally serves
it without payment checks. A hosted catalog should enforce entitlement before
serving premium bundles.
